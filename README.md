# Backup your world before using this!
## Simple-Missing-Replacer


This mod can be useful if you want to remove some mods but don't want to make holes in the terrain.

The mod transforms all blocks from removed mods to nearest non-removed block.

1. Close your save with ores in it.
2. Install this mod.
3. Uninstall ore generating mod.
4. Play your save.

It will replace all occurences of all missing blocks with placeholder in playtime without any lost of perfomance.
